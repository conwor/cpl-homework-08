# Домашнее задание №08 - Общие делители

Прочитать с клавиатуры числа `N` и `M`. Вывести через пробел по увеличению все общие делители
чисел `N` и `M`. После последнего числа символа пробела быть не должно.

Вывод должен заканчиваться переводом строки.

## Пример

### Входные данные

```
30 40
```

### Выходные данные

```
1 2 5 10

```
